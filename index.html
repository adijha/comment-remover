Comment Remover

A tool that allows you to remove the comments from your code.

Is particularly useful when working with large JSON files, because I tend to comment them out a lot, this tools allows me to strip the comments before validation.

As far as my testing goes it works with all the languages that use forward slashes as comments (i.e : it doesn't work with Ruby).

I hope it will help, PRs welcome, especially regarding design/UX. :)

Born out of desperation when I could not find anything similar that allowed me copy, paste, and do ctrl+a on the code.
